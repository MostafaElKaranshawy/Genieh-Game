#### Collect Mode 

     ♦ You are the Good Pharoah and You should Collect All Genieh Coins Without Getting Attacked by Bad Pharoah
     ♦ Mode Have 3 Levels:
        • Easy
          ○ The game has only one Bad Pharoah
        • Medium
          ○ The game has two Bad Pharoahs
        • Hard
          ○ The game has three Bad Pharoahs
